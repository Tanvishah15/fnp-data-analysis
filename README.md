# Sales Analysis Dashboard

## Project Overview
This project focuses on building an interactive **Sales Analysis Dashboard** to analyze sales performance and derive actionable business insights. The dashboard consolidates key metrics related to revenue, orders, customers, products, occasions, time trends, and geography to support data-driven decision-making.

---

## Objectives
- Analyze overall sales performance using key KPIs  
- Identify high-performing products, categories, and cities  
- Understand customer spending and delivery efficiency  
- Track revenue trends across time, occasions, and order hours  
- Enable interactive analysis using filters and slicers  

---

## Key Metrics (KPIs)
- **Total Revenue:** ₹5,86,176  
- **Average Customer Spend:** ₹3,520.98  
- **Average Order to Delivery Time:** ~5.7 days  
- **Total Orders:** Aggregated across selected filters  

---

## Dashboard Insights

### Revenue Analysis
- Revenue is driven primarily by **Soft Toys**, followed by **Sweets** and **Colors**
- Occasion-based sales allow comparison across events such as birthdays, festivals, and anniversaries

### Time-Based Trends
- Monthly revenue trends highlight seasonal demand spikes
- Hourly analysis identifies peak ordering times, supporting operational planning

### Product Performance
- Top revenue-generating products include **Magnam Set**, **Excitation Pack**, and **Expedita Gift**
- Product-level insights help identify best sellers and optimization opportunities

### Geographic Insights
- Cities such as **Kavali, Machilipatnam, Kota, and Bhastpur** show higher order volumes
- Regional insights support logistics and targeted marketing strategies

---

## Interactivity
The dashboard includes:
- Order Date and Delivery Date filters  
- Occasion-based slicers  
- Dynamic charts that update based on selections  

This enables users to drill down into specific periods, locations, or occasions.
